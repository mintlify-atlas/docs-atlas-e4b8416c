## Get started in 3 steps

<Steps>
  <Step title="Install the CLI">
    Run directly without installing, or add to your PATH permanently:

    ```bash
    # Run without installing
    uvx --from git+https://github.com/wuyoscar/gpt_image_2_skill gpt-image -p "a cat astronaut"

    # Or install globally
    uv tool install git+https://github.com/wuyoscar/gpt_image_2_skill
    ```
  </Step>
  <Step title="Set your API key">
    Export your OpenAI API key, or add it to `~/.env`:

    ```bash
    export OPENAI_API_KEY=sk-...
    # or add OPENAI_API_KEY=sk-... to ~/.env
    ```
  </Step>
  <Step title="Generate your first image">
    Run any prompt from the gallery or write your own:

    ```bash
    gpt-image -p "a photorealistic convenience store at 10pm" --size 1k --quality high -f store.png
    ```

    The output path is printed on stdout. Exit code `0` means success.
  </Step>
</Steps>

<Note>
  The CLI reads `OPENAI_API_KEY` from your environment or `~/.env`. Exit codes: `0` success · `1` API/moderation error · `2` bad arguments or missing key.
</Note>
